# Smriddhi Gupta | Portfolio

A responsive, accessible personal portfolio website built with React and Vite, featuring a validated contact form, dark mode, and scroll/hero animations.

## Framework
React (Vite) with React Router for page navigation.

## Features
- Home, About, and Contact pages with client-side routing
- Hero section with profile photo, tagline, and quick stats
- Featured Projects section with links to GitHub repositories
- About page with education timeline, animated skill bars, certifications, and languages
- Contact form with client-side validation (required fields, email format, minimum message length), accessible error messages, and a success confirmation
- Dark mode toggle with preference saved via localStorage
- Fully responsive layout with a mobile navigation menu
- Semantic HTML and accessibility features throughout (labels, alt text, keyboard navigation, visible focus states)

## Setup & Run Instructions

1. Clone or download this repository.
2. Install dependencies:
npm install
3. Start the development server:
npm run dev
4. Open the local URL shown in the terminal (e.g. `http://localhost:5173`) in your browser.
npm run build
